# `<repository-root>/.844196/` (個人スクラッチディレクトリ)

`<repository-root>/.844196/` は、ユーザー固有の補助的なファイル (調査レポート、プラン文書、ハンドオーバー、スクリプト等) を置く個人スクラッチディレクトリ。

ユーザーの global gitignore で除外されているので、プロジェクト側の `.gitignore` を汚さずに置ける。

## 既に存在する場合

過去のセッションで蓄積された文書・ログ・スクリプトが残っている可能性がある。タスクの文脈になりそうなら読み出して参照する。

## 書き出す場合

該当する用途のファイルを書き出すときは `<repository-root>/.844196/` 以下に置く。プロジェクトにまだ無ければ新規に作る。

サブディレクトリは目的別に分けて、直下にバラバラに置かない。既存のサブディレクトリに合致するものがあればそれを使い、無ければ目的を表す新しいサブディレクトリを作る。

よくある用途:
```
<repository-root>/.844196/
  plans/        # 実装計画・設計メモ
  reports/      # 調査・分析のレポート
  handovers/    # セッションハンドオーバースキルが書き出すハンドオーバー文書
  logs/         # 実験のために書き出したログ
  scripts/      # ユーザーが個人で使う補助スクリプト
```
