# Bash ツール実行履歴を参照する

Bash ツール実行履歴は atuin によって記録されており、カスタムエージェントエージェントおよびスキルによって検索可能です。

## どんなときに使うか

会話ログ/トランスクリプトにはコマンドの引数・順序・終了コードが省略されることがあり、atuin の方が情報が濃い。次のような場面で使う:

- 現セッションのコマンド実行サマリを別文書に同梱したい
- 特定の Claude Code セッション ID が手元にあり、そのセッションで実行されたコマンド (引数・終了コード・時刻・cwd) を確認したい
- 断片的なコマンド文字列から、それを実行した Claude Code セッション ID を逆引きしたい
- 時間範囲・cwd・終了コードで俯瞰したい

## カスタムエージェントとスキルの使い分け

- `atuin:recall-commands`

  e.g. 現セッションの直前数件を引く、終了コードを 1 件だけ確認する、1 回限りの軽量な確認

- `atuin:command-history-analyzer`

  e.g. 検索範囲が広い、複数クエリの組合せ、時間範囲俯瞰・複数セッションの突合

  atuin の生出力は数千〜数万行になり、メインで直接叩くとその出力がメインコンテキストに乗り続けてコンテキストウィンドウを圧迫してしまう。カスタムエージェントを介することで生出力を閉じ込め、メインには整理されたサマリだけ返る。
